**Infinity Tech** is East Africa's premier technology startup based in **Degmada Hodan, Mogadishu, Somalia**. We engineer native iOS & Android mobile applications, responsive web platforms, and multi-tenant School SaaS software for over 50+ educational institutions and enterprise clients across Somalia.

---

## 🌟 Key Features & Solutions

### 1. 📱 Native Mobile App Development
- **Frameworks**: Native iOS & Android development built with **Flutter, Swift, Kotlin, and React Native**.
- **Features**: Offline data caching engine, mobile money payment gateway integrations, push notifications, and biometric authentication optimized for 4G/5G mobile networks in Somalia.
- **Store Publishing**: Guaranteed Apple App Store & Google Play Store release compliance.

### 2. 🌐 Web Application Development & Custom UI/UX
- **Tech Stack**: Powered by **Node.js, Express, JavaScript, and React**.
- **Performance**: Guaranteed sub-2 second load speeds, full mobile/tablet responsiveness, search engine optimization (SEO), and custom CMS administrative control panels.

### 3. 🎓 All-In-One School SaaS Management Platform
- **Automation**: Automates student enrollment, monthly tuition fee invoicing, parent WhatsApp/SMS alerts, teacher gradebooks, attendance tracking, and report cards.
- **Cloud Security**: Multi-tenant cloud database architecture with 99.9% uptime SLA and automated daily encrypted backups powering 50+ Somali schools.

### 4. 🤖 DeepSeek AI Assistant & WhatsApp Integration
- **AI Chatbot**: Integrated AI assistant powered by the **DeepSeek API** (`deepseek-chat`) delivering concise, professional text responses and project photo attachments.
- **WhatsApp Escalation**: Direct one-click connection to human support specialists on WhatsApp (**+252 615 565 249**).

## 🚀 Quick Start & Installation

### Prerequisites
- [Node.js](https://nodejs.org/) (v18.0.0 or higher)
- npm (v9.0.0 or higher)

### Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/hashitheog/infinitytech.git
   cd infinitytech
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm start
   ```
   Or directly with Node:
   ```bash
   node server.js
   ```

4. **Access the application**:
   Open your browser and navigate to `http://localhost:3000`.
